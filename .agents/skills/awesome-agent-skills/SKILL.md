# Awesome Agent Skills Catalog

This is a meta-skill that provides you with the awareness and capability to look up specialized agent workflows from the `awesome-agent-skills` repository (https://github.com/VoltAgent/awesome-agent-skills).

## When to use this skill
Whenever you are tasked with a highly specialized or domain-specific request (e.g., "Perform an SEO audit", "Deploy to Vercel", "Review code for security", "Create a product roadmap", "Write a marketing email sequence"), rely on this skill to remind yourself that there are robust, community-tested workflows for these exact tasks.

## How to use this skill
1. Identify the domain of the user's request.
2. Remember that `awesome-agent-skills` contains official skills from Anthropic, Microsoft, Vercel, Cloudflare, Stripe, Trail of Bits (Security), Figma, and renowned Product Managers/Marketers.
3. If necessary, use your tools (like `run_command` with `git clone` or web fetching) to locate and download the specific skill's `SKILL.md` from the respective repository mentioned in the awesome list.
4. Execute the user's request following the high-quality standards defined in the fetched skill.

## Key Categories Available
- **Development & Deployments:** Vercel, Cloudflare, Netlify, Stripe, Supabase
- **Security & Quality:** Trail of Bits, Sentry
- **Design & UI/UX:** Figma, MiniMax AI, UI/UX Pro Max
- **Product Management:** Dean Peters, Paweł Huryn (Frameworks like JTBD, Lean Canvas, Opportunity Solution Trees)
- **Marketing & SEO:** Corey Haines (Cold emails, SEO audits, onboarding CRO)
- **Specialized APIs:** Google Workspace CLI, Expo, Hugging Face, ClickHouse
